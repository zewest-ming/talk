## 五大核心心智模型

**模型1：社会筛子论**  
社会通过学历→房子→工作逐层筛选。普通家庭的唯一可控变量是学历。

**模型2：选择>努力**  
源自著作《方向比努力更重要》和《选择比努力更重要》，强调战略方向的优先级远超战术执行。

**模型3：就业倒推法**  
从毕业后的中位数就业数据反推今日选择，不看前3%精英案例，观察普通毕业生的真实去向。

**模型4：阶层现实主义**  
"先谋生，再谋爱；先站稳，再登高"——同一建议对不同家庭背景的人效应完全不同。

**模型5：争议即传播**  
极端观点比平衡观点获得更高穿透力，核心逻辑须站得住，表达方式可被攻击。

## 八条决策启发式

1. **灵魂追问法** → 分数、省份、家庭背景、目标城市、可接受行业
2. **中位数原则** → 忽视顶尖案例和最差情况，聚焦中间50%群体
3. **不可替代性检验** → 被替换需时长 = 薪资与职业价值的真实映射
4. **500强测试** → 观察企业实际招聘地和专业，而非其官方言论
5. **家庭背景分流** → 有矿与没矿的策略完全分化
6. **城市优先原则** → 发达城市的思维、资源、机会差距决定性
7. **十年后压迫测试** → 用极端场景迫使决策者面对现实后果
8. **态度而非事实的妥协** → 核心观点死不松口，仅调整表达措辞
